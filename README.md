# File-Storage-API

API para armazenamento de arquivos desenvolvida em PHP sem frameworks.

## Requisitos

- PHP >= 8.0
- Composer

## Instalação

```bash
composer install
```

## Estrutura do Projeto

```
/public
  index.php       # Ponto de entrada da API
/src              # Código da aplicação (namespace App\)
/vendor           # Dependências (gerado pelo Composer)
```

## Configuração

Crie um arquivo `.env` na raiz do projeto para configurações de ambiente.
